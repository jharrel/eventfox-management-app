# Project Manager App 

Ruby on Raills app that helps you organize tasks and projects for you and your team. 

## Installation

1. Clone this repo and run `bundle install`
2. Run `rake db:migrate`
3. Run `rake db:seed`
4. Run `rails s`
4. Open your browser to `localhost:3000`
